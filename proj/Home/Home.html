Keyboard Shortcuts
Global
  • F1 Go to File tab + File Manager
  • F2 Go to File tab + File Search
  • F3 Go to RDP tab
  • F4 Go to AI tab
AI Panel
  • Tab Toggle sidebar
  • 1 / 2 / 3 Switch Chat / Terminal / Brow subtab
  • ↑ / ↓ Move through session list (sidebar open)
  • → Jump to notification session
  • ← Return to previous session
  • Shift+N New terminal (Terminal subtab, sidebar open)
  • Shift+D Delete current terminal session
Inside Terminal
  • Enter Send input (Shift+Enter for newline)
  • Tab Apply autocomplete suggestion
  • Esc Close autocomplete
  • ↑ / ↓ Navigate autocomplete, or move cursor when input is empty
  • Ctrl+T Scroll to bottom
  • F6 Toggle SUPER (auto-approve) + focus input bar
